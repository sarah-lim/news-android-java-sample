# news-android-java-sample
News Daily a simple android application using NewsAPI of https://newsapi.org/

## Project Goal
Write an application written in Java using the following: 
- MVP pattern
- Databinding
- Navigation

## Dependencies
- Timber 5.0.1
- Retrofit 2.9.0
- Dagger 2.28.1
- rxAndroid rxJava 3.0.0
- okhttp 4.9.1
- gson 2.8.8
- glide 4.12.0
